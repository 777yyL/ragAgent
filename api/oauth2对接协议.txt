状态码：302

state、menu_uri、

1.1.2 获取access_token 
接口地址：https:// ip:port/artemis/api/application/auth/v2/app/oauth2/token 
请求方式：POST 
Content-type: application/x-www-form-urlencoded 
请求参数： 
参数名 数据类型 参数说明 备注 
client_id string API网关注册的appKey  
client_secret string API网关注册的appSecret  
grant_type string 授权类型 固定值authorization_code 
redirect_uri url字符串 回调地址 注意需要将url进行URLEncode 
code string 授权码 由获取授权码接口返回。 
 海康平台Oauth2服务接口说明书           
4 
 
返回属性： 
参数名 数据类型 参数说明 备注 
access_token string 授权令牌 授权服务器返回给第三方应用的授权
令牌。 
refresh_token string 刷新令牌 access_token快过期时，可以使用该值
进行刷新 
expires_in int 授权令牌
的有效期 
固定值1800，单位为秒，即此
access_token会在30分钟内过期。 
返回示例： 
{ 
 "code": "0", 
 "msg": "SUCCESS", 
 "data": { 
  "access_token": " f78d3082b9ee4864bc32bb8a37f87c95", 
  "refresh_token": " f78d3082b9ee4864bc32bb8a37f87c95", 
  "expires_in": 1800 
 } 
}
